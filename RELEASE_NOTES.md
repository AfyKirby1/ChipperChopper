## 📥 **Installation**

### 📋 **Requirements**
- Minecraft 1.21.4
- Fabric Loader 0.16.9+
- Fabric API 0.119.3+1.21.4
- Java 21+

### 🚀 **Quick Install**
1. Download `chipper-chopper-1.0.0.jar` from releases
2. Place in `.minecraft/mods/` folder
3. Launch Minecraft with Fabric
4. Press 'O' key to activate!
